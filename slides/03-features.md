### Code Highlighting

```javascript
function createPresentation() {
  const slides = ["Introduction", "Content", "Conclusion"];

  return slides.map((slide) => ({
    title: slide,
    content: "Amazing content here!",
  }));
}
```

Notes: Code blocks are syntax highlighted automatically.

### Math Support

You can include mathematical equations:

$$E = mc^2$$

$$\sum_{i=0}^n i^2 = \frac{(n^2+n)(2n+1)}{6}$$

Notes: Math equations are rendered using KaTeX.
